# 🚀 Duolingo Korean Quick Select 사용 가이드

## 📦 설치 완료 후 확인 방법

### 1단계: 확장 프로그램 아이콘 확인
브라우저 우측 상단에 **초록색 "ㄱ" 아이콘**이 나타납니다.

### 2단계: 듀오링고 레슨 페이지로 이동
- https://www.duolingo.com/lesson 에서 한국어 학습을 시작하세요
- 단어 선택 문제가 나올 때까지 진행합니다

### 3단계: 확장 프로그램 아이콘 클릭
- 우측 상단의 "ㄱ" 아이콘을 클릭하세요
- 팝업 창이 열리면 다음을 확인할 수 있습니다:
  - ✅ **상태**: 활성화됨 / 비활성
  - ⌨️ **현재 입력**: 실시간 초성 입력 표시
  - 📊 **통계**: 총 입력 횟수, 자동 선택 횟수

### 4단계: 초성 입력 테스트

문제에 다음 단어들이 있다고 가정:
```
우리는  파티  학과에서
```

**테스트 1**: `ㅇ` 입력
- "우리는"과 "학과에서"가 파란색으로 하이라이트
- 팝업의 "현재 입력"에 `ㅇ` 표시

**테스트 2**: `ㅇㄹ` 입력  
- "우리는"만 정확히 일치 → 녹색으로 변경
- 0.2초 후 자동 선택
- 팝업의 "자동 선택 횟수" 1 증가

**테스트 3**: `ㅍ` 입력
- "파티"만 일치 → 즉시 자동 선택

## 🔧 문제 해결

### Q: 팝업에서 "비활성" 상태로 표시됩니다
**A**: 현재 페이지가 듀오링고 레슨 페이지가 아닙니다.
- `https://www.duolingo.com/lesson` 으로 이동하세요
- 페이지를 새로고침(F5) 하세요

### Q: 키를 눌러도 반응이 없습니다
**A**: 다음을 확인하세요:
1. 문제 화면에 단어 버튼들이 표시되어 있나요?
2. 입력 창(input)이 활성화되어 있지 않나요? (빈 곳을 클릭하세요)
3. 개발자 콘솔(F12)에서 에러가 있는지 확인하세요

### Q: 특정 단어가 선택되지 않습니다
**A**: 
- 해당 단어가 `lang="ko"` 속성을 가지고 있는지 확인하세요
- 페이지 구조가 변경되었을 수 있습니다 (GitHub에 이슈를 남겨주세요)

## 💡 팁

1. **빠른 입력**: 초성을 빠르게 연속으로 입력할 수 있습니다
2. **실수 수정**: Backspace로 마지막 초성만 지울 수 있습니다
3. **전체 초기화**: ESC를 누르면 입력을 모두 지웁니다
4. **통계 확인**: 팝업에서 얼마나 효율적으로 사용했는지 확인하세요

## 🎯 활용 시나리오

### 시나리오 1: 긴 단어 빠르게 선택
문제: "학교에서", "학생", "학과"
- `ㅎㄱㅇ` 입력 → "학교에서" 자동 선택

### 시나리오 2: 비슷한 단어 구분
문제: "먹다", "먹었다", "먹을"
- `ㅁ` 입력 → 3개 모두 하이라이트
- `ㅁㄱ` 입력 → "먹다", "먹었다" 하이라이트  
- `ㅁㄱㄷ` 입력 → "먹다"만 정확 매칭 → 자동 선택

### 시나리오 3: 실수 수정
- `ㅎㄱㅅ` 입력 (잘못 입력)
- Backspace → `ㅎㄱ`
- `ㅈ` 추가 → `ㅎㄱㅈ`

## ✨ 영어 버튼과의 차이점

| 특징 | 영어 | 한글 (이 확장) |
|------|------|----------------|
| 입력 방식 | 알파벳 직접 입력 | 초성 입력 |
| 예시 | "tea" → t, te, tea | "우리" → ㅇ, ㅇㄹ |
| 하이라이트 | 파란색 | 파란색 (일치), 녹색 (정확) |
| 자동 선택 | 완전 일치 시 | 초성 완전 일치 시 |
