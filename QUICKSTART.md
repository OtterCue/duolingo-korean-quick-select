# ⚡ 빠른 시작 가이드

## 📥 설치 (30초)

1. **ZIP 파일 압축 해제**
2. **Chrome에서** `chrome://extensions/` 접속
3. **"개발자 모드"** 켜기 (우측 상단)
4. **"압축해제된 확장 프로그램을 로드합니다"** 클릭
5. 압축 해제한 폴더 선택
6. ✅ 완료!

## 🎮 사용법 (10초 학습)

### 듀오링고 레슨 시작

1. 단어 선택 문제가 나오면
2. 페이지 빈 곳을 클릭 (포커스 해제)
3. 한글 초성 입력: `ㅇ`, `ㄴ`, `ㄷ` ...
4. 파란색으로 하이라이트!
5. 하나만 남으면 자동 선택!

### 예시

```
문제: "We are here together" → 한국어로?

초성 입력:
ㅇ → "우리는", "여기" 파란색 ✨
ㅇㄹ → "우리는"만 녹색 → 자동 선택! 🎯
```

## ⌨️ 단축키

- **ㄱ, ㄴ, ㄷ, ...** - 단어 검색
- **Backspace** - 한 글자 지우기  
- **ESC** - 초기화

## 🔧 안 되면?

### 1. 콘솔 열기
- **F12** 키 누르기
- **Console** 탭 선택

### 2. 다음 메시지 확인
```
✅ Duolingo Korean Quick Select 활성화됨!
✅ 단어 은행 발견!
```

### 3. 안 보이면
- `chrome://extensions/`에서 오류 확인
- 페이지 새로고침 (F5)
- 확장 프로그램 재설치

## 💡 팁

### 자주 쓰는 초성
- `ㅇㄹ` = 우리는
- `ㅇㄱ` = 여기  
- `ㄱㅌ` = 같이
- `ㅇㅅ` = 있어요

### 주의사항
- ✅ 단어 **선택** 문제에서만 작동
- ❌ 타이핑 문제에서는 작동 안 함
- ⚠️ 입력창이 아닌 빈 곳 클릭 후 사용
